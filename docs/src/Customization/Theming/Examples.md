# Examples

## Repositioning

-   **Right**

    ```css title="theme.css"
    #song-container {
        right: 0;
        transform-origin: right;
    }
    ```

-   **Bottom left**

    ```css title="theme.css"
    #song-container {
        bottom: 0;
    }
    ```

-   **Bottom right**

    ```css title="theme.css"
    #song-container {
        bottom: 0;
        right: 0;
        transform-origin: right;
    }
    ```
