If disabled, then only tabs in the background and all tabs of unfocused windows (disabled due to a bug on Chrome) are searched for media.

This is disabled by default. The reason for this is that if a tab is focused and the window is focused, then it will probably be shown (on stream) anyway and there's no reason to show duplicate information.
Filter Mode

Filters

A filter will check every tab's url. Regular filters check the hostname part of the url (The hostname of https://maps.google.com/maps is maps.google.com). Its functionality is best illustrated by an example. google.com will filter every host containing the text google.com so https://maps.google.com/something but also https://fakegoogle.com/. Regex filters will check if a regular expression matches the whole url.
